# 3. 亜子と小宮の生活空間・環境

## 索引(Index)と階層

- 福岡県福岡市
    - 小宮の住居、兼仕事場
    - 亜子のバイト先のカフェ
    - 亜子の実家

## 詳細

小宮の住居、兼仕事場:
- 概要：小宮と亜子が同棲している部屋、兼、小宮の仕事場
- ロケーション：福岡県福岡市中央区
- 呼称：単に「家」「部屋」
- 構造：防犯が機能しているミドルレンジの賃貸マンション（広めのワンルーム）
- 雰囲気：半分仕事場・半分住居
- ワンルームの構造:
    - 少し広め
    - ソファベット：窓際にある
    - ベット：生活動線から少し離れている
    - ミニテーブル：ソファベットの近くにある
    - デスク
    - キッチン（カウンターなし）
- ワンルームに存在する物質:
    - 「青い世界」（完成品）：青い空と青い海で構成されたジグソーパズル → 壁に設置
    - 亜子が音楽制作するためのノートパソコンと録音機器
    - 小宮が仕事で使うパソコンとモニターディスプレイ
- ワンルームに存在しない物質:
    - テレビ

亜子のバイト先のカフェ:
- 概要：亜子がバイトしているカフェ
- ロケーション：福岡県福岡市中央区
- 呼称：単に「カフェ」

亜子の実家:
- 概要：亜子の実家
- ロケーション：福岡県福岡市
